### 如何运行

> 注意：请先部署 [NodeJs](https://nodejs.org/en/) 和 [Gatsby](https://www.gatsbyjs.com/) 环境。

- 克隆项目

```shell
git clone https://github.com/cloudups/shifeiqi.top.git
cd shifeiqi.top
```

- 安装依赖

```
npm install
```

- 运行

```
npm run start
```
